配置说明

数据路径语法：

  • field_name - 访问对象的字段
  • nested.field - 访问嵌套对象的字段
  • array[0] - 访问数组的第一个元素
  • items[0].name - 访问数组第一个元素的name字段

数据转换类型：

  • 乘法运算：将数值乘以指定因子，参数：{"factor": 1.5}
  • 除法运算：将数值除以指定除数，参数：{"divisor": 100}
  • 四舍五入：保留指定小数位数，参数：{"digits": 2}
  • 格式化：使用格式字符串，参数：{"format": "{:.2f}°C"}
  • 正则提取：使用正则表达式提取，参数：{"pattern": "\\d+"}

示例配置：

{
  "temperature": 25.6,
  "humidity": 60.2,
  "device": {
    "id": "DEV001",
    "status": "online"
  },
  "sensors": [
    {"name": "temp1", "value": 25.6},
    {"name": "hum1", "value": 60.2}
  ]
}

映射配置：
- 字段名：temperature，路径：temperature，类型：浮点数
- 字段名：device_id，路径：device.id，类型：字符串
- 字段名：first_sensor，路径：sensors[0].value，类型：浮点数
