# DT企业智能管理系统

一个基于Django开发的企业级智能管理系统，包含用户管理、部门管理、生产管理、文件管理、客户关系管理、财务管理等20+功能模块，集成AI智能服务，为企业提供全面的数字化解决方案。

## 核心功能详解

### 基础管理模块

#### 1. 用户管理 (User Management)
- **用户注册与登录**: 完整的用户身份验证系统，支持多因素认证，包括用户名密码登录、验证码验证等安全机制
- **权限管理**: 基于角色的访问控制(RBAC)，支持细粒度权限分配，可针对具体功能、数据进行权限控制
- **组织架构**: 多层级用户管理体系，支持用户分组和部门归属，可构建复杂的组织结构树
- **个人信息管理**: 用户个人资料、头像、联系方式等信息维护，支持自定义字段扩展
- **安全设置**: 密码修改、登录日志查看、账户安全监控，保障账户安全
- **用户状态管理**: 支持用户启用/禁用、锁定等状态控制，便于人员管理

#### 2. 部门管理 (Department Management)
- **部门结构**: 支持无限层级部门组织架构，可构建集团-公司-部门-小组等多级结构
- **员工分配**: 灵活的员工与部门关联管理，支持一人多部门、主部门设置
- **部门权限**: 部门级别的资源访问控制，实现数据隔离和权限管理
- **部门信息**: 部门基本信息、负责人、描述等维护，支持部门编码管理
- **部门统计**: 部门人员统计、业务数据汇总等分析功能

#### 3. 岗位管理 (Position Management)
- **职位体系**: 职位等级、职级定义和管理，构建完整的职位晋升通道
- **岗位职责**: 岗位工作内容和责任范围定义，明确岗位职责边界
- **任职管理**: 员工岗位分配和变更记录，支持历史记录追溯
- **岗位编制**: 岗位编制管理，控制各部门人员配置
- **岗位权限**: 岗位级别权限设置，实现基于岗位的权限控制

#### 4. 基础数据管理 (Basic Data Management)
- **地区管理**: 省市区三级地理信息维护，支持多级地址联动选择
- **职级体系**: 企业内部员工职级标准定义，支持职级晋升管理
- **职务管理**: 公司职务体系和职务权限配置，明确职务权责
- **字典数据**: 系统通用数据字典维护，支持自定义数据字典项
- **行业分类**: 客户、供应商等行业分类标准维护
- **数据同步**: 基础数据同步机制，确保数据一致性

### 协同办公模块

#### 1. 个人办公 (Personal Office)
- **日程管理**: 个人日程安排、提醒设置和日历视图，支持重复事件、提醒通知
- **工作记录**: 日报、周报、月报等周期性工作记录，支持模板化报告
- **任务管理**: 个人任务清单、优先级设置、进度跟踪，支持子任务分解
- **通讯录**: 内部员工联系信息查询和管理，支持部门分组查看
- **个人中心**: 个人信息维护、偏好设置、消息通知，个性化工作环境
- **工作统计**: 个人工作量统计、任务完成情况分析

#### 2. OA系统 (Office Automation)
- **消息通知**: 系统内消息、公告、邮件通知，支持消息分类和已读状态
- **审批流程**: 自定义审批流程、在线审批处理，支持会签、或签等审批模式
- **会议管理**: 会议室预订、会议安排、会议纪要，支持会议室资源管理
- **文档管理**: 企业文档库、文档版本控制、文档审批，支持在线预览和编辑
- **资产管理**: 固定资产登记、维修、报废管理，支持资产生命周期跟踪
- **车辆管理**: 公车使用申请、调度、维护记录，支持车辆使用统计
- **印章管理**: 印章申请、使用记录、归还管理，确保印章使用安全
- **考勤管理**: 员工考勤记录、请假申请、加班统计

#### 3. 项目管理 (Project Management)
- **项目创建**: 项目立项、基本信息录入、项目编号生成，支持项目模板
- **进度跟踪**: 甘特图、里程碑、任务分解结构(WBS)，实时进度监控
- **成员管理**: 项目团队组建、角色分配、权限控制，支持外部协作人员
- **预算管理**: 项目预算制定、成本控制、费用统计，支持预算预警
- **文档协作**: 项目文档共享、版本管理、在线编辑，支持文档评论和@提醒
- **风险管理**: 风险识别、评估、应对措施跟踪，降低项目风险
- **资源管理**: 项目资源分配、资源冲突检测、资源利用率分析
- **报告生成**: 项目周报、月报自动生成，支持自定义报告模板

#### 4. 任务管理 (Task Management)
- **任务分配**: 任务创建、指派、优先级设置，支持批量任务分配
- **进度跟踪**: 实时进度更新、完成状态标记，支持进度条可视化
- **团队协作**: 多人协作任务、子任务分解，支持任务依赖关系
- **时间管理**: 工时统计、任务耗时分析，支持时间跟踪功能
- **任务提醒**: 任务截止提醒、逾期提醒，确保任务按时完成
- **任务模板**: 常用任务模板管理，提高任务创建效率

### 业务管理模块

#### 1. 客户管理 (Customer Relationship Management)
- **客户信息**: 客户基本资料、联系信息、企业信息维护，支持客户标签管理
- **联系人管理**: 客户方联系人信息、职位、关系维护，支持主要联系人标识
- **跟进记录**: 客户拜访记录、沟通历史、跟进计划，支持跟进提醒
- **客户公海**: 公共客户资源池、客户领取、分配机制，优化客户资源利用
- **客户分类**: 客户等级、行业分类、来源渠道管理，支持多维度客户分类
- **线索管理**: 销售线索收集、分配、转化跟踪，提高销售转化率
- **商机管理**: 销售机会跟踪、阶段管理、赢单概率评估
- **合同管理**: 客户合同关联、履约情况跟踪、合同到期提醒
- **订单管理**: 客户订单录入、状态跟踪、回款管理
- **数据分析**: 客户画像、价值分析、趋势预测，辅助销售决策
- **客户关怀**: 客户生日提醒、节日问候、定期回访计划

#### 2. 生产管理 (Production Management)
- **工序管理**: 基本工序定义、工序集配置、工艺路线设计，支持工序标准工时管理
- **设备管理**: 生产设备档案、维护保养、运行状态监控，支持设备故障记录
- **生产计划**: 生产任务下达、排程计划、产能分析，支持计划调整和优化
- **BOM管理**: 物料清单维护、版本控制、成本核算，支持BOM多级结构
- **数据采集**: 生产数据实时采集、统计分析，支持多种数据采集方式
- **质量管理**: 质量标准设定、检验记录、质量报告，支持质量异常处理
- ** sop管理**: 标准作业程序维护、版本控制、执行跟踪
- **生产统计**: 产量统计、效率分析、成本核算，支持生产绩效评估

#### 3. 合同管理 (Contract Management)
- **合同信息**: 合同起草、审批、签署、归档全流程，支持电子签名
- **产品管理**: 合同关联产品、服务内容定义，支持产品配置管理
- **付款管理**: 合同付款计划、回款记录、账期管理，支持逾期提醒
- **履约跟踪**: 合同执行状态监控、里程碑管理，确保合同履约
- **合同模板**: 标准合同模板库、快速生成合同，提高合同起草效率
- **合同变更**: 合同条款变更记录、审批流程，确保变更合规
- **合同提醒**: 合同到期提醒、付款提醒、关键节点提醒
- **合同统计**: 合同金额统计、履约情况分析、客户合同分布

#### 4. 财务管理 (Financial Management)
- **报销管理**: 费用申请、审批、打款全流程，支持多种费用类型
- **发票管理**: 发票开具、作废、回款跟踪，支持发票验真功能
- **财务审批**: 财务相关审批流程处理，支持多级审批机制
- **账务管理**: 收支记录、账目核对、财务对账，确保账务准确性
- **预算管理**: 部门预算制定、使用情况跟踪、预算预警
- **数据分析**: 财务报表、成本分析、预算执行情况，支持图表展示
- **税务管理**: 税务申报提醒、税务资料管理、税务风险预警
- **资金管理**: 资金流水记录、银行账户管理、资金计划制定

### 技术支撑模块

#### 1. 文件管理 (Document Management)
- **文件存储**: 文件上传下载、存储空间管理，支持大文件分片上传
- **文件夹管理**: 目录结构组织、权限控制，支持文件夹模板
- **文件分享**: 内外部文件共享、链接分享，支持分享权限控制
- **版本控制**: 文件版本管理、历史版本追溯，支持版本对比
- **权限管理**: 细粒度文件访问权限控制，支持按用户、角色、部门授权
- **在线预览**: 支持文档、图片、音视频等多种格式在线预览
- **搜索功能**: 全文检索、标签搜索、高级筛选
- **回收站**: 文件删除保护、回收站管理、文件恢复

#### 2. 系统管理 (System Management)
- **系统配置**: 系统参数设置、功能开关控制，支持配置项分组管理
- **模块管理**: 系统功能模块启用/禁用，支持模块依赖关系管理
- **日志管理**: 操作日志、系统日志查看分析，支持日志导出和审计
- **数据备份**: 定期数据备份、恢复机制，支持多种备份策略
- **性能监控**: 系统运行状态监控、性能分析，支持实时监控面板
- **安全管理**: 系统安全策略配置、访问控制、安全审计
- **接口管理**: 系统API接口管理、访问权限控制、调用统计

#### 3. 权限体系 (Permission System)
- **角色管理**: 系统角色定义、权限分配，支持角色继承和权限组合
- **菜单管理**: 系统菜单配置、访问控制，支持菜单层级和权限关联
- **数据权限**: 行级数据权限控制，支持数据范围限制
- **操作权限**: 功能操作权限细粒度控制，支持按钮级别权限
- **权限审计**: 权限分配记录、权限变更历史，确保权限管理合规
- **权限继承**: 支持基于组织架构的权限继承机制

#### 4. 数据统计 (Data Analytics)
- **仪表板**: 关键指标可视化展示，支持自定义仪表板布局
- **报表生成**: 各类业务报表自动生成，支持定时任务和手动触发
- **图表分析**: 多维度数据图表展示，支持柱状图、折线图、饼图等多种图表
- **数据导出**: 数据导出为Excel、PDF等格式，支持导出模板配置
- **数据钻取**: 支持从汇总数据钻取到明细数据
- **实时分析**: 实时数据监控、异常数据预警
- **自定义报表**: 支持用户自定义报表字段、筛选条件、展示方式

### 环境要求
- Python 3.8+
- Django 4.0+
- MySQL 5.7+ 或 PostgreSQL 10+

### 安装步骤
