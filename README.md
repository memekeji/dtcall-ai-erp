# DT企业智能管理系统

一个基于Django 5.2开发的企业级智能管理系统，包含用户管理、客户管理、生产管理、财务管理、项目管理、OA办公等21个核心功能模块，集成AI智能服务，为企业提供全面的数字化解决方案。

## 技术架构

### 后端技术栈
- **框架**: Django 5.2.1 + Python 3.8+
- **数据库**: PostgreSQL 10+ (支持PostgreSQL 10及以上版本)
- **缓存**: Redis 6+ (用于会话缓存和系统缓存)
- **API**: Django REST Framework + SimpleJWT认证

### 前端技术栈
- **框架**: LayUI 2.8+
- **图表**: ECharts
- **文件处理**: Office文档在线预览支持

### 第三方集成
- **云存储**: 阿里云OSS、腾讯云COS、百度云BOS、七牛云Kodo、AWS S3、WebDAV
- **AI模型**: OpenAI、Azure OpenAI、Anthropic、Google Gemini、百度文心、阿里通义、腾讯混元、DeepSeek、字节豆包、本地模型
- **语音服务**: 语音转文字(STT)支持多种服务
- **工业协议**: MQTT、Modbus、OPC UA等工业数据采集协议

## 核心功能模块

## 快速开始

### 环境要求
- Python 3.8+
- Django 5.2+
- PostgreSQL 10+
- Redis 6+

### 安装步骤

1. 克隆项目
```bash
git clone <repository_url>
cd dtcall
```

2. 创建虚拟环境
```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

3. 安装依赖
```bash
pip install -r requirements.txt
```

4. 配置数据库
确保PostgreSQL数据库已创建，并配置正确的连接信息

5. 配置环境变量
```bash
# 设置Django密钥
export DJANGO_SECRET_KEY='your-secret-key'

# 数据库配置(可选，使用settings.py中的默认值)
export DATABASE_HOST='192.168.1.149'
export DATABASE_PORT='5432'
export DATABASE_NAME='dtcall'
export DATABASE_USER='dtcall'
export DATABASE_PASSWORD='dtcall'

# Redis配置(可选，使用settings.py中的默认值)
export REDIS_HOST='192.168.1.149'
export REDIS_PORT='6379'
```

6. 运行迁移
```bash
python manage.py migrate
```

7. 启动服务
```bash
python manage.py runserver 0.0.0.0:8000
```

### 生产部署建议

- 使用Gunicorn作为WSGI服务器
- 配置Nginx作为反向代理和静态文件服务器
- 启用HTTPS
- 配置定期数据库备份
- 监控系统运行状态
